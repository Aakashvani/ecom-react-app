# Kimaye Clone 



Hassle free online fruit ordering platform in Mumbai & Delhi, Kimaye is a promise of consistently delivering the safest and superior quality fruits.

This website is our Unit 5 Masai Construct week project.
## Tech Stack

**Client:** React, Redux 

**Server:** Node, Express

**Deployment:** Vercel
